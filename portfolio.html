    Roulette Tafel - Frontend

    De roulette tafel is een project die ik heb gemaakt voor een les uit het eerste jaar 'Inleiding programmeren' waar ik voor het eerst Javascript gingen schrijven. De opracht was om een code te schrijven die minimaal een Array, een function(event) met een if/else statement en een eventListner moesten hebben.

    Voor deze les had ik nog nooit Javascript geschreven, maar ik wou wel een uitdaging aan gaan, omdat ik al best was kennis over HTML en CSS had. Daarom koos ik er ook voor om een groter project aan te gaan, namelijk een Roulette tafel. Dit was erg buiten mijn comfortzone, maar erg leuk om mee te puzzelen!
